# MiniProyecto

Ejercicio práctico para comprender el uso de JavaScript dentro del desarrollo web frontend.  
Durante este proyecto se trabajarán conceptos fundamentales relacionados con manipulación del DOM, consumo de APIs y persistencia de datos en el navegador.

---

# Instrucciones

## 1. Realizar un fork del repositorio

Haz clic en el botón **Fork** dentro de GitHub para crear una copia del repositorio en tu cuenta personal.

---

## 2. Clonar el repositorio en tu computadora

Ejecuta el siguiente comando dentro de la carpeta donde deseas guardar el proyecto:

```bash
git clone URL_DEL_REPOSITORIO
```

---

## 3. Acceder a la carpeta del proyecto

```bash
cd nombre-del-repositorio
```

---

## 4. Abrir el proyecto en Visual Studio Code

```bash
code .
```

---

## 5. Guardar cambios en Git

Agregar archivos modificados:

```bash
git add .
```

Crear un commit:

```bash
git commit -m "Descripción breve de los cambios"
```

---

## 6. Subir cambios al repositorio

```bash
git push origin main
```

---

# Secuencia de contenidos

Durante el desarrollo del proyecto se trabajarán los siguientes temas:

- Manipulación del DOM
- Fetch API
- WebStorage API

---

# Recomendaciones

- Mantener una estructura ordenada de archivos.
- Utilizar nombres descriptivos para variables e ids.
- Documentar partes importantes del código mediante comentarios.
- Realizar commits frecuentes durante el desarrollo.

---

# Objetivo general

Comprender cómo JavaScript interactúa con HTML y CSS para crear interfaces web dinámicas e interactivas sin necesidad de frameworks frontend.
